Linways Faculty Evaluation – Autofill

Install the userscript below (via Tampermonkey) to auto-fill all teachers' evaluations with Excellent, Very Good, or Random (4–5).

⬇ Install Userscript

If Tampermonkey is installed, the link opens an install prompt automatically.

1) Install Tampermonkey

  • Chrome Web Store
  • Edge Add‑ons
  • Firefox Add‑ons

2) Install the Script

Click the big button above. Tampermonkey will show an Install screen—confirm it.

3) Use on Linways

Open the teacher list page on Linways. A floating panel appears at the bottom‑right. Use it to run:

  • Auto All → Excellent
  • Auto All → Very Good
  • Auto All → Random

Tip: Press Alt+L to show/hide the panel, or use Tampermonkey → "Toggle Panel (Alt+L)".

Having trouble? Ensure Tampermonkey is installed and enabled. Then refresh Linways and try again.
